User Interface
==============

The user interface provides abstractions for the command line, GUI,
and graphics.

    1. A common way to parse commands
    2. GUI abstraction (QML/Qt/HTML5/jQuery).
    3. Graphics abstraction (WebGL/OpenGL/SVG).
