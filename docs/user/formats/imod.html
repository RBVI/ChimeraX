IMOD Segmentation Files

IMOD is a suite of programs for processing and analyzing data from electron microscopy and tomography. IMOD binary files (format imod, filename suffix .imod or .mod) contain segmentation mesh data and contour data.

See also: color map, color zone, volume mask, Segment Map, Measure and Color Blobs

IMOD segmentation files can be opened using the File menu or the open command:

Usage: open  filename [ format imod ] [ meshes  true | false ] [ contours  true | false ]

By default, only the mesh data are read and displayed (meshes true and contours false). Each mesh object is opened as a surface model. Setting contours true indicates reading the contours and adding them as markers and links. Each contour object is opened as a marker model.
