EMDB Segmentation File Format (SFF)

EMDB segmentation files are specified as format sff or by filename suffix .sff (XML) or .hff (HDF5). These files may contain:

  • a map or “lattice” of integer-valued region identifiers, sometimes with colors and descriptions for the regions
  • surfaces or “meshes”

See also: color map, color zone, volume mask, segmentation, Segment Map, Measure and Color Blobs

EMDB segmentation files can be opened using the File menu or the open command:

Usage: open  filename [ format sff ] [ maxSurfaces  N ] [ debug  true | false ]

The EMDB-SFF lattice is opened as a volume dataset with a surface for each region, using colors from the file, if available. The maxSurfaces option limits the number of surfaces created (default 100), and the debug option specifies detailed information to the Log as the file is read.
