Toolbar: Nucleotides

Icons in the Nucleotides tab of the Toolbar allow changing the style of nucleotides in atomic models. Style changes are only visible for nucleotide atoms that are displayed, such as with the Molecule Display icon or the show command. Some of the nucleotide icons also appear in the Molecule Display tab. See also: Basic Actions, Presets menu, making images

Clicking an icon shows the executed command in the Log. Nucleotide style icons act on the currently selected atoms, if any (specified as sel), otherwise the currently displayed atomic models (specified as a list of model numbers).
