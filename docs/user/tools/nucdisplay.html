Toolbar: Nucleotides

Icons in the Nucleotides tab of the Toolbar allow changing the display of nucleotides in atomic models. Some of the nucleotide icons also appear in the Molecule Display tab. See also: Basic Actions, Presets menu, making images

Clicking an icon shows the executed command in the Log. The selAtoms specifier indicates the current atomic selection, or if nothing is selected, all nucleotide residues.
