Tool: Map Statistics

Map Statistics reports the minimum, maximum, and mean values, the standard deviation (SD) from the mean, and the root-mean-square (RMS) deviation from zero for volume data (map) models.

Map Statistics can be started from the Volume Data section of the Tools menu. It does not have a graphical interface, but simply runs the measure mapstats command on the currently displayed map models (acting on selected maps only, if any, otherwise all). If a map is cropped to a subregion and/or shown at step > 1, only the corresponding values are used to calculate the statistics. See also: Volume Viewer, measurements
