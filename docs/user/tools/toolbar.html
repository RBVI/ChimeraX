Tool: Toolbar

The Toolbar across the top of the ChimeraX window contains icons for performing actions with a single click. Clicking an icon shows the executed command in the Log. See also: Basic Actions

Like other tools, the Toolbar can be started from the Tools menu (more...). It is organized into several tabbed sections:

  • Home – will be customizable, currently populated automatically:   [...]

  • Molecule Display: 

  • Nucleotides: 

  • Graphics: 

  • Map: 

  • Markers – button-function assignments related to markers: 

  • Right Mouse – button-function assignments: 

Each tab is further organized into sections with names. The section names and descriptions of individual icons (buttons) can be hidden and shown by unchecking and checking the following options in the Toolbar context menu:

  • Show button labels
  • Show section labels

Icon sections that would extend beyond the current window width can be disclosed by clicking the double arrowhead at the right end of the Toolbar. The tabs are all part of the same tool and cannot be dragged apart, unlike tabs for separate tools that were previously dragged on top of each other (see panel manipulation).
