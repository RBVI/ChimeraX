Tool: Scenes

Scenes allows saving and restoring ChimeraX scenes. A scene currently includes:

  • model and clipping plane positions (as in view)
  • atom, bond, and pseudobond display states, styles, colors, and sizes
  • 2D labels and arrows
  • volume data (map) display styles, levels, coloring, and masking (such as with surface dust)
  • background color

**In progress: not all aspects are saved; more will be added added as development proceeds**

Users should not delete or close any models or their parts that appear in a scene, as trying to restore the scene will not regenerate them. Scenes are saved in sessions. The Scenes tool is also implemented as the scenes command. See also: making images, making movies

Scenes can be started from the Depiction section of the Tools menu and manipulated like other panels (more...). Currently saved scenes are shown as thumbnail images. Clicking the Add button saves the current contents of the graphics window as a new scene with the specified name, if any, otherwise an automatically generated integer name. The thumbnails can be reordered by drag-and-drop.

Clicking a thumbnail chooses the scene, restores it, draws an outline around its thumbnail, and places its name in the entry field. Clicking a blank area of the dialog unchooses the scene, removes the outline, and clears the name entry field.

When a thumbnail is chosen, the following actions become available:

  • Update – overwrite the chosen scene with the current ChimeraX contents
  • Rename – enter a new name for the chosen scene
  • Delete – delete the chosen scene
