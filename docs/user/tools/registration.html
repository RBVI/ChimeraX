Tool: Registration

A user is prompted to register after several days of using ChimeraX on a given device. Registration is optional, free of charge, and recommended to help us understand how the software is used. See also: registration status

Like other tools, Registration can be opened from the Tools menu (more...).

The location of the registration file can be reported with command:

info path user unversioned data
