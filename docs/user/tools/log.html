Tool: Log

The Log tool echoes commands that have been executed, with any truncations expanded. It also shows messages and other output information, potentially including hyperlinks and command-execution links (for example, to generate an assembly or to show the sequence of a chain). Restoring a session restores its Log contents as a collapsible section with a disclosure triangle. See also: Log preferences, Command History, ChimeraX browser, log, usage, model information, measurements

Like other tools, the Log can be opened from the Tools menu (more...). Its context menu includes:

  • Insert Image – include a thumbnail image of the current display
  • Save As... save contents to an HTML file
  • Clear – remove all contents
  • Copy Selection
  • Select All
  • Raise Log When Logging Occurs – whether to show the Log (bring to front if tabbed behind another panel, or reopen if closed) whenever some action or message adds to its contents; can be set as a preference
  • Executable Command Links – whether command links should be executable or show help (saved as a preference)
