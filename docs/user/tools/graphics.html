Tool: Graphics

The Graphics tool provides a series of icons for adjusting or acting on the overall display with a single click. Like other tools, this interface can be opened from the Tools menu and closed by clicking the small “x” near its top right corner, or shown/hidden with the toolshed command. See also: Molecule Display icons, Density Maps icons, Mouse Modes for Right Button icons, making images, making movies

Clicking an icon shows the executed command(s) in the Log.
