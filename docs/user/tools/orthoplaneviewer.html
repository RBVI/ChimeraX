Tool: Orthoplane Viewer

The Orthoplane Viewer is a tool for viewing axial, coronal, or sagittal slices of 3D volume data such as medical images or density maps in a separate window. Volume data can be opened with the command open (which automatically starts Volume Viewer). Display color and thresholds (intensity/value mapping) can be adjusted with Volume Viewer or the volume command.

See also: ui view, ChimeraX DICOM Reference, Download DICOM, DICOM in VR tutorial, DICOM segmentation tutorial, ChimeraX for Medical Image Analysis

Even without data, Orthoplane Viewer can be started from the Volume Data section of the Tools menu. The Orthoplane Viewer window can be manipulated like other panels in ChimeraX (more...).

The Model menu on the top right allows choosing between the currently open volume models. The menu on the top left allows choosing the slice direction as Axial, Coronal, or Sagittal.

At the bottom of the window, there is a slider for navigating through the data planes along the corresponding axis. Alternatively, the following keys can be used, optionally + Shift for faster movement:

  • letters A and D
  • left and right arrow keys

The depth of the current plane is reported to the right of the slider.

The slice view does not rotate, but zooming and translation follow the default mousemode assignments.
