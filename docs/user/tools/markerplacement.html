Tool: Marker Placement

Marker Placement sets parameters for markers and links created with the mouse, as specified with Markers icons or the mousemode command. See also: Volume Viewer, measure length, vseries, color, size

Marker Placement can be started from the Volume Data section of the Tools menu or by clicking any of the Markers icons:

Like other tools, it can also be started with the command toolshed show and moved, resized, etc.

Settings:

  • Marker color – a color well (click to change), default
      radius [default 1.0]
  • Link color – a color well (click to change), default
      radius [default 0.5]
  • Link new marker to selected marker (default off) – whether to link markers as they are created

Pressing enter (return) after entering a new radius value is required for it to take effect. A change in color or radius applies to any currently selected markers and links, as well as those which are subsequently created.
