Tool: Marker Placement

Marker Placement sets parameters for markers and links created with the mouse, as specified with Markers icons or the mousemode command. See also: Volume Viewer, measure length, vseries

Marker Placement can be started from the Volume Data section of the Tools menu or by clicking any of the icons in the Markers toolbar:
Like other tools, it can also be started with the command toolshed show and moved, resized, etc.

Settings:

  • Marker color – a color well (click to change), default
      radius [default 1.0]
  • Link color – a color well (click to change), default
      radius [default 0.5]
  • Link new marker to selected marker (default off) – whether to link markers as they are created
