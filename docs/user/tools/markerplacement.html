Tool: Marker Placement

Marker Placement sets parameters for creating markers and links with the mouse, as specified with Markers icons or the mousemode command. Markers and links can also be created and adjusted with the marker command. Markers placed on volume series data are assigned a frame attribute with value indicating the associated time step within the series. See also: Volume Viewer, measure length, vseries, the tape measure mouse mode

Marker Placement can be started from the Volume Data section of the Tools menu or by clicking any of the icons in the Markers tab of the Toolbar: 
The tool window can be moved, resized, etc.

Settings:

  • Marker color – a color well (click to change), default
      radius [default 1.0]
  • Link color – a color well (click to change), default
      radius [default 0.5]
  • Link new marker to selected marker (default off) – whether to link markers as they are created

Pressing enter (return) after entering a new radius value is required for it to take effect. A change in color or radius applies to any currently selected markers and links, as well as those which are subsequently created.
