Toolbar: Map

Icons in the Map tab of the Toolbar act on volume models such as density maps. They apply to the displayed models only (those with    checkboxes in the Model Panel checked for themselves and any parents). See also: Medical Image icons, making images

Clicking an icon shows the executed command in the Log. Map icons act on the currently selected volume models, if any (specified as sel), otherwise the currently displayed volume models (specified as a list of model numbers). There may be alternative commands for performing the same action.
