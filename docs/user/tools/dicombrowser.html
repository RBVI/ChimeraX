Tool: DICOM Browser

DICOM files contain associated information (metadata) that can be displayed in the DICOM Browser and the related DICOM Metadata tool. See also: Download DICOM, ChimeraX DICOM Reference

The DICOM Browser can be shown by choosing a model in the Model Panel that corresponds to a patient or study (read from DICOM) and clicking the Info button. It can also be started from the Medical Imaging section of the Tools menu, although if no DICOM data are open at the time, it will be blank.

The DICOM Browser dialog shows the DICOM patient→study→series hierarchy, with selected descriptors at each level. One or more rows can be chosen (highlighted) by clicking and dragging with the left mouse button; Ctrl-click (or command-click if using a Mac) toggles whether a row is chosen.

Full DICOM Metadata for the chosen row(s) can be shown by using the context menu to Load Metadata from Chosen Entries, or by clicking the Show Metadata button at the bottom of the DICOM Browser dialog.
