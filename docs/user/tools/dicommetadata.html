Tool: DICOM Metadata

DICOM files contain associated information (metadata) that can be displayed in the DICOM Metadata viewer and the related DICOM Browser tool. See also: Download DICOM, ChimeraX DICOM Reference

DICOM Metadata contains a full list of DICOM series attributes and can be shown by either of the following:

  • choosing one or more series in the Model Panel and clicking the Info button
  • using the context menu in the DICOM Browser tool to Load Metadata from Highlighted Entries; if shown for a patient or study, the attributes of all of the corresponding series will be included.

Columns in the DICOM Metadata window include tags, attribute names, and attribute values. Doubleclicking a row opens an explanation (if available) in the Help Viewer for that specific attribute at dicomlookup.com.
