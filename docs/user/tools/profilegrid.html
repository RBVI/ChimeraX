Profile grid of the seed alignment for the PFAM 7 transmembrane receptor (Secretin family) associated with a structure of the glucagon-like peptide-1 receptor, PDB 7LCJ. Two residues with 100% type conservation are selected and labeled.

Tool: Profile Grid

Profile Grid shows a condensed view of a multiple sequence alignment in which the rows are the possible sequence characters (residue one-letter codes and gap), the columns are positions in the alignment (as in the standard view of a sequence alignment), and the values in the cells are percentages of that type of residue at that position.

Much like the Sequence Viewer, the Profile Grid tool interacts with any associated structures, and it can include headers above the sequence data.

Opening a sequence alignment file with the open command option viewer grid shows it as a Profile Grid.

Multiple Profile Grid windows can be shown at the same time, and they can be manipulated like other panels in ChimeraX (more...).
