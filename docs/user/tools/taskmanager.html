Tool: Task Manager

The Task Manager interface lists background calculations (also called jobs or tasks) launched from ChimeraX, such as a Blast Protein sequence search. The Task Manager can be started from the Utilities section of the Tools menu and manipulated like other panels (more...). Jobs can also be listed and killed with the taskman command. See also: Modeller Comparative, Model Loops

Checkboxes in the bottom section of the panel control which columns of information are shown in the table of jobs, with buttons:

  • All – show all columns
  • Default – restore the previously saved preference for which columns are shown
  • Standard – use “factory default” set of initially displayed columns
  • Set Default – save the currently shown set as a user preference
  • Toggle Controls – show/hide the checkbox section

Clicking a column header sorts by the values in that column.

Clicking Clear Finished removes the completed jobs from the list. Close dismisses the tool, whereas Help opens this page in the Help Viewer.

Available columns of information:

  • ID (Local) – job ID on the local machine, used to specify a job to the taskman command
  • ID (Webservices) – unique string assigned to a job running as a web service hosted by the UCSF RBVI
  • Task – job name
  • Started – job start time
  • Runtime – time since start
  • Status – job status
