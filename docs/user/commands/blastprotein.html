Command: blastprotein

Usage:
blastprotein ( chain-spec | sequence-spec ) [ database  pdb | nr ] [ matrix  similarity-matrix ] [ cutoff  evalue ] [ maxSeqs  M ] [ log  true | false ] [ toolId  N ]

The blastprotein command runs a protein sequence similarity search using a BLAST web service hosted by the UCSF Resource for Biocomputing, Visualization, and Informatics (RBVI). It is the command implementation of the Blast Protein tool. One use is to search with a target sequence of unknown structure to find templates for comparative modeling.

The query sequence can be taken from a protein structure or a sequence alignment open in ChimeraX:

  •  chain-spec
    – OR –
  •  sequence-spec

The protein sequence database to search can be:

  • pdb (default) – experimentally determined structures in the Protein Data Bank (PDB)
  • nr – NCBI “non-redundant” database containing GenBank CDS translations + PDB + SwissProt + PIR + PRF excluding environmental samples from whole-genome sequencing; this database is much larger than pdb alone and takes much longer to search

The matrix option indicates which amino acid similarity-matrix to use for alignment scoring (uppercase or lowercase can be used):

  • BLOSUM45
  • BLOSUM50
  • BLOSUM62 (default)
  • BLOSUM80
  • BLOSUM90
  • PAM30
  • PAM70
  • PAM250
  • IDENTITY

The cutoff evalue is the maximum or least significant expectation value needed to qualify as a hit (default 1e-3). Results can also be limited with the maxSeqs option (default 100); this is the maximum number of unique sequences to return; more hits than this number may be obtained because multiple structures or other sequence-database entries may have the same sequence.

When results are returned, the hits are listed in a Blast Protein window. The names of hits from the pdb database (a subset of nr), are links that can be clicked to fetch the structure, and if a structure chain was used as the query, to automatically superimpose the hit and query structures with matchmaker. The names of other hits are GI numbers that can be clicked to show the corresponding web pages in the NCBI Protein database. Checkboxes allow designating one or more hit sequences to show as a multiple alignment with the query. Many other types of information can be shown and used to sort the hits: alignment scores, structure resolution, ligand residue names, etc. (details...)

The log option indicates whether to also list the results in the Log (default false).

The toolId option allows supplying an integer identifier for a specific instance of the Blast Protein tool, allowing multiple panels for different searches to be shown simultaneously. If only one set of results is shown at a time, this option is not needed.
