Command: registration status

Usage:
registration  status [ verbose ]

A user is prompted to register after several days of using ChimeraX on a given device. Registration is done using a graphical form, the Registration tool. Registration is optional, free of charge, and recommended to help us understand how the software is used.

The registration status command reports in the Log whether ChimeraX is registered on the device, and if so, the expiration date. If ChimeraX is not registered on the device, the total number of uses and the number of days of use are reported. The verbose option shows all registration information, if any, or lists the days of use.
