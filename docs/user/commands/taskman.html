Command: taskman

Usage:
taskman list

Usage:
taskman kill  local-ID

The taskman list command lists any ongoing background calculations (also called jobs or tasks) in the Log. See also: Task Manager

The list information includes a local job ID number that can be used to abort a specific job with taskman kill, for example:

taskman kill 3
