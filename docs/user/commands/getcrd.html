Command: getcrd

Usage:
getcrd  atom-spec  [ model | scene | screen ]

The getcrd command reports the coordinates of the specified atoms in the Log. See also: distance, save PDB, camera, view, Map Coordinates

The keywords indicate which coordinate system to use:

  • model - the original coordinate system of a model
  • scene (default) - the scene coordinate system, which includes changes from moving any models relative to other models (by fitting, matching, etc.)
  • screen - the screen coordinate system, with X horizontal increasing rightward, Y vertical increasing upward, Z perpendicular to the screen increasing toward the viewer, and origin at the viewpoint

If models have not been moved relative to one another, the model and scene options are the same. The scene and camera options both preserve spatial relationships among models that have been moved relative to one another.
