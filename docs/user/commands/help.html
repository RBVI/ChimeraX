Command: help

Usage:
help [ newTab ]  command

The command help shows the help page for the specified command, if available, in the ChimeraX browser. The newTab keyword (which cannot be truncated and must appear before command) indicates that the page should be opened in a new tab. If the command exists but lacks a help page, its usage information is shown in the Log instead. See also: open

Examples:

help set
help volume gaussian
