Command: scenes

Usage:
scenes save [ name ]

Usage:
scenes rename  name   new-name

Usage:
scenes restore  name

Usage:
scenes delete  name

Usage:
scenes list

The scenes command allows saving and restoring ChimeraX scenes. A scene currently includes:

  • model and clipping plane positions (as in view)
  • atom, bond, and pseudobond display states, styles, colors, and sizes
  • cartoon/ribbon styles including worms
  • 2D labels and arrows
  • volume data (map) display styles, levels, coloring, and masking (such as with surface dust)
  • background color

**In progress: not all aspects are saved; more will be added added as development proceeds**

Users should not delete or close any models or their parts that appear in a scene, as trying to restore the scene will not regenerate them. Scenes are saved in sessions. Thumbnail images and names of the existing scenes are shown in the Scenes graphical interface. See also: view, making images, making movies, Animations in ChimeraX tutorial (2025)

The current scene can be saved and named with scenes save:

  • if name is omitted, an integer name will be automatically generated
  • if the name of an existing scene is supplied, that scene will be overwritten with the current contents

An existing scene can be renamed with scenes rename, restored with scenes restore, or deleted with scenes delete. The command scenes list reports the names of the existing scenes in the Log.
