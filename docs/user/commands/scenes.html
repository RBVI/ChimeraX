Command: scenes

Usage:
scenes save [ name ]

Usage:
scenes restore  name

Usage:
scenes delete  name

Usage:
scenes rename  name   new-name

Usage:
scenes list

The scenes command allows saving and restoring ChimeraX scenes, which include model and clipping plane positions (as in view), as well as display states, colors, 2D labels, etc.

**In progress: not all aspects are saved; more will be added added as development proceeds**

Scenes are saved in sessions. Thumbnail images and names of the existing scenes are shown in the Scenes graphical interface. See also: view, making images, making movies

The current scene can be saved and named with scenes save:

  • if name is omitted, an integer name will be automatically generated
  • if the name of an existing scene is supplied, that scene will be overwritten with the current contents

An existing scene can be renamed with scenes rename, restored with scenes restore, or deleted with scenes delete. The command scenes list reports the names of the existing scenes in the Log.
