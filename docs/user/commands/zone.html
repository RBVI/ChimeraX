Command: zone

Usage:
zone  residue-spec  [ residueDistance  dr ] [ labelDistance  dl ] [ surfaceDistance  ds ] [ label  true | false ] [ ribbons  true | false ]

Usage:
zone clear

Usage:
zone setting [ labelColor  color-spec | default ] [ labelBackgroundColor  color-spec | none ] [ save  true | false ]

The zone command shows atomic detail and residue labels near a specified single residue of interest. Density-map or other volume data displays can also be restricted to a zone near that residue, and different cutoff distances can be used for the atoms, residue labels, and volume displays:

  • residueDistance (initial default 5 Å) – for all-atom display of nearby residues
  • labelDistance (initial default 4 Å) – for labels on nearby residues
  • surfaceDistance (initial default 8 Å) – for volume data displays (surface, mesh, or solid); does not affect molecular surfaces

The cutoff values remain in effect for automatic application with the zone command and/or zone mouse mode until they are explicitly changed or ChimeraX is exited.

The label option indicates whether to show zone residue labels (default true; false removes all residue labels). The ribbons option shows the beyond-zone residues of the same atomic model as a narrow ribbon (default true; false does not modify any existing cartoons).

The zone clear command clears the labels and returns to what was shown before zoning.

The zone setting command allows changing:

  • labelColor – label text color; if specified as default or auto, the label color will automatically adjust to white or black depending on the color of the label background, or if there is no label background, depending on the color of the window background
  • labelBackgroundColor – color of the rectangular backgrounds shown behind the labels (initial default none, meaning no backgrounds)

By default (save true), these changes are saved as preferences for zone labels only (not other labels). Enabling virtual reality automatically changes the zone-label color to yellow
and zone-label background color to black without saving these values as preferences.

For molecular surfaces, see surface zone. See also: volume zone (with newMap false), label, show, cartoon, select zone, zone atom specification
