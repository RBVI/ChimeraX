Command: runscript

Usage:
runscript  script.py  arguments

The runscript command allows running a Python script with pathname script.py  and one or more command-line arguments. Substituting the word browse for the pathname brings up a file browser window for choosing the name and location interactively. The arguments are passed to the Python script as sys.argv[1], sys.argv[2], etc.. The --runscript startup option calls this command. See also: open
