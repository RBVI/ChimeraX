Command: delete

Usage:
delete [ atoms | bonds | pseudobonds ]  spec  [ name  pseudobond-model-name ]

The delete command deletes the specified atoms (default), bonds, or pseudobonds (synonym pbonds). If the object-type keyword is given, the spec is allowed to be blank. Bonds and pseudobonds can be specified by their flanking atoms, or as a selection with sel. Pseudobonds can also be specified by model number, certain built-in classifications, or pseudobond model name as shown in the Model Panel. See also: hide, close
