Command: set

Usage:
set [ keyword  value ]N

The set command sets various global parameters. If given without arguments, it reports the current settings in the Log.

  • bgColor - background color
  • silhouettes - whether to outline borders
  • silhouetteColor - outline color
  • silhouetteWidth - outline thickness
  • silhouetteDepthJump - change in depth needed to trigger drawing an outline
  • subdivision - fineness of facets in stick, balls, spheres

One or more parameters can be specified in a single command. Examples:

set bg sky blue
set sil true silhouettew 2

A vertical bar “|” designates mutually exclusive options, and default values are indicated with bold. Synonyms for true: True, 1. Synonyms for false: False, 0.

• set bgColor  color
Set background color, where color can be a color specification by name, hexadecimal color code, or component values (default black).
• set silhouettes  true | false
Whether to show silhouettes, outlines that emphasize borders and discontinuities.
• set silhouetteColor  color
Set silhouette color, where color can be a color specification by name, hexadecimal color code, or component values (default black).
• set silhouetteWidth  linewidth
Set silhouette linewidth in pixels (default 1).
• set silhouetteDepthJump  depth
Set depth change needed to trigger drawing silhouettes (default 0.03 Å).
• set subdivision  level
Set subdivision level, where a higher value increases the apparent smoothness of stick, ball-and-stick, and sphere representations by increasing the number of planar facets used to approximate them (default 1).
