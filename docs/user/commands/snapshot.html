Command: snapshot

Usage:
snapshot [ directory  save-location ]

The snapshot command saves an image file just like the Toolbar icon: , with the initial default location as follows:

save ~/Desktop/imageN.png supersample 3

(N = 1,2, ...)

A different save location can be specified with the directory option. The specified location is stored in the preferences and will apply to subsequent uses of both the snapshot and spin-movie icons: . The word browse can be substituted for save-location to raise a file browser window for specifying the location interactively. See also: save, saving images, saving movies
