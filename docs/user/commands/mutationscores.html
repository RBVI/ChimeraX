Command: mutationscores

The mutationscores command shows the results of deep mutational scanning on protein structures and as interactive 2D plots. Deep mutational scanning generally involves mutating each position in a protein to all 20 standard amino acids and assessing the resulting large number of mutants with multiple high-throughput assays. Each assay yields a score, and a scatter plot of the mutants can be drawn to show their scores for any given pair of assays.

The scores are read from a mutation scores .csv file (details...) that can be opened from the the File menu or with the open command. Each file gives rise to a mutation set, and multiple mutation sets can be open at the same time. The data can be associated with an atomic structure by using the chain option of the open command, or later by using mutationscores structure. See also: ChimeraX visualization of mutation scores

  • mutationscores scatterplot – plot mutants by two types of score
  • mutationscores define – define a new score based on existing scores
  • mutationscores undefine – remove a previously defined score
  • mutationscores histogram – show a histogram of score values
  • mutationscores label – show color-coded residue labels on the associated structure
  • mutationscores statistics – compute mutation score averages and standard deviations
  • mutationscores umap – project the 20-dimensional vector of the scores of the 20 residue types at a position into 2D with UMAP
  • mutationscores structure – associate the data with a specified structure chain
  • mutationscores list – list the currently open sets of mutation data
  • mutationscores close – close a set of mutation data

The mutationSet option allows specifying which set to use when more than one is open. If only one is open, the option is not needed. The names are derived from the input filenames (for example, abcg2 if read from abcg2.csv). The names of open sets can be listed in the Log with mutationscores list, and a set can be closed with mutationscores close.

• mutationscores list
List the names of the currently available score sets in the Log. The names are derived from the input filenames (for example, abcg2 if read from abcg2.csv). A set can be closed with mutationscores close.
• mutationscores close [ mutation-set-name ]
Close a specified set of mutation scores. The names of currently open sets can be listed with mutationscores list.
