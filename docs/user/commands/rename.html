Command: rename

Usage:
rename  model-spec [ new-name ] [ id new-id ]

The command rename changes the name of the specified model(s) to new-name and/or the ID number of a single specified top-level model to new-id. The new-name should precede the id option if both are given, and if new-name includes spaces, it should be enclosed in quotes. If only the name is changed, the Model Panel will not show the change until its display is updated by some other action such as opening or closing a model or clicking a checkbox. See also: open

Example:

rename #1 mutant4
rename #5 id #3
rename #8 "closed conformation" id #5
