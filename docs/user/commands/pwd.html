Command: pwd

Usage:
pwd

The command pwd reports the current working directory in the Log. The working directory can be changed with the cd command or by choosing File... Set Working Folder from the menu. See also: open, save

Upon startup with the ChimeraX icon, the initial working directory is ~/ (user's home directory) on Linux and ~/Desktop on Windows and Mac. Startup from the system command line uses the current shell directory instead.
