Command: snfg

Usage:
snfg [ show | hide ] [ structures  atom-spec ] [ size  angstroms ] [ atoms  true | false ]

The snfg command displays carbohydrate residues using 3D-SNFG (Symbol Nomenclature for Glycans) representation, where each monosaccharide is shown as a colored geometric shape placed at the sugar ring centroid.

SNFG is a community standard for depicting glycan structures using geometric symbols with specific colors:

Shape Sugar Class Examples
Sphere Hexoses Glucose (blue), Mannose (green), Galactose (yellow)
Cube N-acetylhexosamines GlcNAc (blue), GalNAc (yellow)
Diamond Hexuronic acids, Sialic acids GlcA (blue), Neu5Ac (purple)
Cone Deoxyhexoses Fucose (red), Rhamnose (green)
Star Pentoses Xylose (orange), Arabinose (green)
Rectangle Di-deoxyhexoses Olivose, Tyvelose
Hexagon Heptoses Kdo (yellow), MurNAc (purple)
Pentagon Ketoses Fructose (green), Apiose (blue)

Colors indicate stereochemistry within each shape class: blue (Glc-type), green (Man-type), yellow (Gal-type), orange (Gul-type), pink (Alt-type), purple (All-type), light blue (Tal-type), brown (Ido-type), red (Fuc/Sia).

Arguments and Options

• snfg [ show ] [ structures  atom-spec ] [ size  angstroms ] [ atoms  true | false ]
Show SNFG symbols for carbohydrate residues. If no structures are specified, all open atomic structures are processed. The size option sets the symbol diameter in angstroms (default 4.0, matching the approximate cross-section of a monosaccharide ring). The atoms option controls whether sugar atoms remain visible (default true). When true, symbols are shown inside the sugar rings alongside the atomic representation. When false, sugar atoms are hidden and only the SNFG symbols are shown with connecting lines between them.
• snfg hide [ structures  atom-spec ]
Hide (remove) SNFG symbols from the specified structures.

Examples

snfg

Show SNFG symbols for all carbohydrates in all open structures.

snfg #1

Show SNFG symbols only for carbohydrates in model #1.

snfg size 6

Show larger (6 angstrom) SNFG symbols.

snfg atoms false

Show SNFG symbols only, hiding the sugar atoms.

snfg hide

Remove all SNFG symbols.

Notes

Sugar residues are identified by their PDB residue names (e.g., NAG, MAN, FUC, GAL). The command recognizes common carbohydrate residue codes from the wwPDB Chemical Component Dictionary.

SNFG symbols are placed at the centroid of the sugar ring (pyranose or furanose). If no ring is detected, the symbol is placed at the residue center of mass.
