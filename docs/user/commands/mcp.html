Command: mcp

Usage:
mcp start [ port ]

Usage:
mcp stop

Usage:
mcp info

Usage:
mcp setup

The mcp command enables ChimeraX to listen for commands from the Claude AI chatbot using the Model Context Protocol developed by Anthropic. The capability for Claude to interact with ChimeraX was first implemented by Alexis Rohou at Genentech, and then incorporated into ChimeraX as the mcp command. See also: remotecontrol, Claude Chat to Operate ChimeraX

  • mcp start [ port ] – start the ChimeraX REST server for MCP bridge connections using port number port (default 8080)
  • mcp stop – stop the ChimeraX REST server
  • mcp info – check to see if ChimeraX is listening for commands via REST server on localhost port 8080
  • mcp setup – write Claude Desktop configuration file to allow it to control ChimeraX using MCP. The file name and location depends on the platform:
    • on Mac: ~/Library/Application Support/Claude/claude_desktop_config.json
    • on Windows: ~/AppData/Roaming/Claude/claude_desktop_config.json
