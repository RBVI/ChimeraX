Command: version

Usage:
version [ verbose | bundles | packages ]

Without keywords, the version command reports the UCSF ChimeraX version number and date of last modification in the Log. With the verbose option, it also gives the time of last modification, the code branch, and last commit.

The bundles option gives the versions of ChimeraX tool bundles in addition to the same information as the verbose option.

The packages option gives python package version numbers in addition to the same information as the verbose option.

See also: log
