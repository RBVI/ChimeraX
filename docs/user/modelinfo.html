Model Information

Certain types of input files include not only 3D data for display, but also associated information that may be shown in the Log when the files are opened. Information can be suppressed by using open with logInfo false. See also: automatic styling

The following may be shown (if available) for an atomic structure:

  • structure title
    • clicking the “more info” link uses log metadata to show title, literature references, nonstandard residues, etc.

  • nonstandard residues
    • clicking the 3-character residue name selects the corresponding residue(s) in the structure
    • clicking the description shows the RCSB Protein Data Bank webpage for that residue type in the ChimeraX browser

  • macromolecular assemblies (this information can also be shown with the sym command)
    • clicking an assembly identifier generates the corresponding assembly using sym defaults, and adjusts the view to contain the assembly

  • biopolymer chain information
    • clicking a chain identifier selects the corresponding chain in the structure
    • clicking a chain description opens the sequence in the Sequence Viewer, associated with all of the corresponding chains in the structure
