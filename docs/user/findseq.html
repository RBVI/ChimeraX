Select Sequence

Choosing Select... Sequence from the menu opens a dialog to find and select a specified sequence in amino acid or nucleotide chains. Sequence patterns to select can also be specified with the select command. The exact sequence or a pattern can be specified:

  • lowercase letters are automatically uppercased
  • regular expressions (as explained for egrep) can be used
  • ambiguity codes are automatically expanded as follows:
    • protein: B matches aspartic acid (D) and asparagine (N), Z matches glutamic acid (E) and glutamine (Q)
    • nucleic acid: R matches the standard purine nucleotides (A, G), Y matches the standard pyrimidine nucleotides (C, T, U), and N matches any of the five (A, C, G, T, U)

To instead display the entire sequence of a chain and then select parts of it with the mouse, use the Sequence Viewer.
