Desktop Support

ChimeraX uses the distribution-provided xdg-utils tools to integrate with the desktop environment. Those tools can add a desktop shortcut, integrate into the desktop menus, and associate types of files with ChimeraX. How well that is done depends on the Linux distribution and the particular desktop environment. Normally, this integration is done by the installer. But it can also be done by the user.

• linux xdg-install [ verbose  true | false ]
Install the desktop support for the user. Does not affect the system's installation, if any.
• linux xdg-uninstall [ verbose  true | false ]
Remove the desktop support for the user. Does not affect the system's installation, if any.
