Computers

Options

  • b2 - Boltz version 2.1.1, actually slightly newer July 8, 2025(?), Mac and Windows no kernels, Mac 32-bit precision, Windows Nvidia bf16-mixed, Linux with kernels (cuequivariance) and bf16-mixed precision.
  • b2a - Boltz version 2.1.1 with ligand affinity
  • b2s - Boltz version 2.1.1 with steering
  • b1 - Boltz-1x version 1.0.0, actually slightly newer from May 28, 2025 Boltz github source code. 32-bit floating point and no diffusion steering.
  • b1s - Boltz 1.0.0 using diffusion steering potentials.
  • b0 - Boltz 0.4.1 from PyPi.
  • Mac M1 Max 32 GB - MacBook Pro 32 core GPU, Tom's laptop
  • Mac M2 Ultra 64 GB - Mac Studio 60 core GPU, descartes.cgl.ucsf.edu
  • Mac M1 16 GB - Mac Mini, 8 core GPU, euclid.cgl.ucsf.edu
  • Linux Nvidia 4090 - VRAM 24 GB, cpu i9-13900K (24 cores) 64 GB (DDR5 5200MHz), minsky.cgl.ucsf.edu
  • Linux i9 CPU - cpu i9-13900K (24 cores) 64 GB (DDR5 5200MHz), minsky.cgl.ucsf.edu
  • Windows i7 CPU - cpu i7-12700K (12 cores) 64 GB DDR5 4000MHz, vizvault.cgl.ucsf.edu
  • Windows Nvidia 3070 - Windows 11, VRAM 8 GB, PCIe 4.0, cpu i7-12700K (12 cores) 64 GB DDR5 4000MHz, vizvault.cgl.ucsf.edu
  • Windows Nvidia 4070 - Windows 10, VRAM 12 GB, PCIe 3.0, cpu i5-6700K (4 cores) 64 GB DDR4 2133MHz, vive.cgl.ucsf.edu
