Tool: Updates

Updates is a tool for listing and installing bundles from the ChimeraX Toolshed. The tool can show:

  • All Available – all bundles/versions available from the toolshed
  • Installed Only – current and available versions of installed bundles only
  • Updates Only – updates available for installed bundles only

Each column can be sorted alphabetically by clicking on the column title.

The Updates tool will pop up automatically when updates to installed bundles are detected. The Toolshed update interval (which also apears in the Toolshed preferences) controls how often to check for updates:

  • every day
  • every week (initial default)
  • every month
  • never

Updates can be started from the Utilities section of the Tools menu and manipulated like other panels (more...). When started from the menu, it defaults to showing all bundles available for the current version of ChimeraX.
