Tool: Fit Ligand

The Fit Ligand tool in ChimeraX runs ligandfit (included in the Phenix nightly build) to fit a small organic molecule such as a ligand within a local region of cryo-EM density. The command implementation is phenix ligandFit. Users should cite:

Macromolecular structure determination using X-rays, neutrons and electrons: recent developments in Phenix. Liebschner D, Afonine PV, Baker ML, Bunkóczi G, Chen VB, Croll TI, Hintze B, Hung LW, Jain S, McCoy AJ, Moriarty NW, Oeffner RD, Poon BK, Prisant MG, Read RJ, Richardson JS, Richardson DC, Sammito MD, Sobolev OV, Stockwell DH, Terwilliger TC, Urzhumtsev AG, Videau LL, Williams CJ, Adams PD. Acta Crystallogr D Struct Biol. 2019 Oct 1;75(Pt 10):861-877.

See also: Fit in Map, Local EM Fitting, Water Placement, Fit Loops, video: Phenix/ChimeraX setup

Phenix must be installed locally and the PhenixUI bundle installed in ChimeraX. The program location can be specified (saved in preferences) beforehand with the phenix location command. If no location has been specified, however, ChimeraX will look for a folder name that starts with phenix in the user's home directory, and on Mac only, if none is found there, then in the system /Applications folder.

Fit Ligand can be started from the Phenix section of the Tools menu.

The Ligand to fit can be specified as one of the following:

  • CCD identifier – idealized structure specified by identifier (residue name) in the PDB Chemical Component Dictionary
  • existing structure – ligand atomic structure already open as a separate model in ChimeraX
  • PubChem identifier – modeled atomic coordinates specified by PubChem compound identifier (CID)
  • SMILES string – modeled atomic structure specified by SMILES string, with coordinates generated using the SMILES translator provided by the National Cancer Institute CADD group

The atomic structure of the Receptor (macromolecular component) and the map to use should already be open in ChimeraX and chosen from the menus of available models; other maps and atomic structures may also be present, although not used in the calculation. The approximate Resolution of the map should also be specified. Additional settings:

  • Center search at
    • picked volume blob – the center will be defined by interactively placing a marker (after OK or Apply is clicked); this uses the mark maximum mouse mode). A dialog will appear for switching the right mouse button between translating the scene and placing this marker. There is also an option to Hide density within [d] Å of existing structure (initially off, with d = 1.8 Å) to make any regions of density not occupied by the receptor easier to see. If the distance value is changed, it is necessary to turn the option off and back on apply the new value. When the marker position is satisfactory, clicking Adjust search zone (if Interactively adjust... was checked) or Start ligand fitting initiates the next part of the calculation.
    • center of model... center of the bounding box of the chosen model
    • center of selection – center of the current selection, e.g., one or more previously created markers
    • center of view – determined as follows:
      1. if center-of-rotation crosshairs are displayed (cofr showPivot true), at the center of the crosshairs;
      2. otherwise, if the center-of-rotation method is fixed, at that center of rotation;
      3. otherwise, if a single map is displayed, in the center of the graphics window in the plane of the window (screen XY coordinates), and at the middle of the map's bounding box in depth (screen Z-coordinate)
      4. otherwise, if more than one map is displayed, as immediately above except using the bounding box of one of the half maps used in the fitting calculation
    • specified xyz position... X, Y, and Z coordinates (Å) relative to the origin of the map and in the coordinate system of the map
  • Search region within [extent] [ligand lengths –OR– angstroms] of center along cell axes (initial default 1.1 ligand lengths)
  • Interactively adjust search region before fitting (initial default on) – whether to allow the user to adjust the rectangular box defining the search space, with initial dimensions as specified in the preceding option; a dialog will appear for switching the right mouse button between moving the example ligand (when selected) to adjust the search center, translating the scene, and adjusting the box size by moving its faces. When the center and box dimensions are satisfactory, clicking Start ligand fitting initiates the calculation.
  • Show H-bonds formed by fit ligand (initial default on) – whether to calculate H-bonds between the ligand and the rest of the structure
  • Show clashes with fit ligand (initial default on) – whether to calculate clashes between the ligand and the rest of the structure

If neither blob-picking or interactive adjustment is chosen, simply clicking Apply on the main dialog (or OK, which also dismisses the dialog) starts the calculation.

Close dismisses the dialog without starting the calculation, whereas Help shows this page in the Help Viewer.
