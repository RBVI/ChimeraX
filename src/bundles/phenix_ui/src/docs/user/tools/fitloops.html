Tool: Fit Loops

The Fit Loops tool in ChimeraX runs fit_loops (included in the Phenix nightly build) to build short peptides such as missing loops into density. The command implementation is phenix fitLoops. Users should cite:

Phenix must be installed locally and the PhenixUI bundle installed in ChimeraX. The program location can be specified (saved in preferences) beforehand with the phenix location command. If no location has been specified, however, ChimeraX will look for a folder name that starts with phenix in the user's home directory, and on Mac only, if none is found there, then in the system /Applications folder.

Fit Loops can be started from the Phenix section of the Tools menu. The existing atomic Structure and the Map to use should already be open in ChimeraX and chosen from the menus of available models; other maps and atomic structures may also be present.

The peptide segments to build or rebuild are specified by selection. The residues of interest can be selected using any method (menu, command, mouseclick in the graphics window, the Gaps table mentioned below, etc.). The residues in a missing segment or gap can be specified by selecting the corresponding missing-structure pseudobond.

Although there is no limit on the total number of residues to model in multiple segments, within an individual segment, modeling ≥15 consecutive residues is not recommended (is more likely to give poor results) and modeling ≥20 is disallowed.

A table of the Gaps in the chosen structure is provided for convenience if the goal is to fill in a missing segment, but it need not be used if some other selection method is preferred. Clicking an entry in the table zooms in on the corresponding region and selects the missing-structure pseudobond. Only one entry can be chosen at a time. Also select N residues on each side of gap allows specifying how many residues on either side of the gap to include in the newly built portion. These end residues are also selected when an entry in the table is chosen, and they are included in the count of consecutive residues to be modeled. Accordingly, the Gap length values in the table will be shown in red when the total length (including the end residues) is too great to model (disallowed), or in orange if allowed but greater than recommended.

Clicking Apply (or OK, which also dismisses the dialog) starts the calculation. Close dismisses the dialog without starting the calculation, whereas Help shows this page in the Help Viewer.

If the Fit Loops calculation gives an unsatisfactory result, the structure can be reverted to its previous state with the undo command or by choosing Edit... Undo from the ChimeraX menu.
