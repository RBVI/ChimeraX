Tool: Tutorial (HTML)

The Tutorial (HTML) tool is the graphical interface to the tutorial commands.

The Global Options section is used to select which atoms are used for computing the center, whether the center calculation is weighted by atomic mass, and whether transformed or untransformed coordinates are used. Clicking the Report Center of Mass button will report the center coordinates in the Reply Log.

The Highlight Options section is used to additionally select the number of atoms to color, and the color to apply. Clicking the Highlight Center of Mass button will set the color of the atoms nearest the center of mass.
